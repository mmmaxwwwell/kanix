Added `body: JSON.stringify({})` to the `createCart()` function in `site/src/lib/cart.ts`. The function was the only POST call in the file that omitted a body, causing Fastify's JSON parser to crash on the empty body (since `Content-Type: application/json` was always set by `apiRequest()`). This matches the pattern used by every other POST call in the same file and what the integration test already does.
