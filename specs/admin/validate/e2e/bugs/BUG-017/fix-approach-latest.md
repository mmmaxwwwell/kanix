Extended the `CartLineWithDetails` interface (API) and `CartLineItem` interface (site) with `isKit`, `kitTitle`, and `kitComponents` fields. In `getCartWithItems`, when `kitInfo` is non-null, the code now fetches the kit definition title and all kit selection variants to populate these fields. The `variantTitle` for kit lines is set to the kit's title (e.g., "Starter Kit") instead of the first variant's title. The checkout page's `renderCart` function now renders kit items with the kit title and a component list, and omits the SKU for kit lines.
