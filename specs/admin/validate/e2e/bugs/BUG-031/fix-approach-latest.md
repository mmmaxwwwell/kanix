Added a Contributor Dashboard card to `account_screen.dart` that is conditionally shown when `contributorDashboardProvider` returns `AsyncData` (indicating the current user is linked to a contributor record). The card contains a `ListTile` with `onTap: () => context.go('/contributor')`. Added `go_router` and `contributor_provider` imports. Non-contributors see no tile because the provider returns a 404 error (AsyncError), so `hasValue` is false.
