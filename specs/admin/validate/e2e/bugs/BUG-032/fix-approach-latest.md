Removed the `githubLinked` gate from `contributor_dashboard_screen.dart`. The screen now directly watches `contributorDashboardProvider` and renders based on its async state: `AsyncLoading` → spinner, `AsyncData` → `_DashboardBody`, `AsyncError` with DioException 404 → "Not a contributor account" message, other errors → generic error text. Removed unused `auth_provider` import, added `dio` import for `DioException` type check.
